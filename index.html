About EXDC

EXDC is a revolutionary token that enables the distributed exchange of any files for any ERC20 token on the Polygon network. Our platform goes beyond simple file exchange, allowing users to create and operate various types of shops without any technical expertise.

EXDC Platforms

EXDC offers multiple platforms to cater to different needs and use cases:

Starpy.me

Starpy.me is our comprehensive shop creation platform that allows anyone to create instant shops without the need for an operator. With Starpy.me, you can:

  • Set up a fully customizable shop in minutes
  • Manage inventory and process orders automatically
  • Accept payments in EXDC tokens and other supported cryptocurrencies
  • Leverage advanced features for a complete e-commerce experience

Cryptofans.io

Cryptofans.io offers a simplified vending machine experience for content creators, particularly those dealing with photos and videos. Our API provides an easy option to create instant vending machines:

  • Quick setup for selling photos and videos
  • Seamless integration with existing platforms
  • Automated content delivery upon payment
  • Perfect for influencers, artists, and content creators

How to Exchange

Exchanging files or creating shops using EXDC is simple:

  1. Connect your wallet to our platform
  2. For file exchange:
    • Upload the file you want to exchange
    • Set the price in EXDC tokens or any other supported ERC20 token
    • Wait for a buyer to purchase your file
  3. For shop creation:
    • Choose between Starpy.me for full shops or Cryptofans.io for simple vending machines
    • Follow the platform's easy setup process
    • Add your products or content
    • Start selling and earning EXDC tokens
  4. Receive tokens directly in your wallet once transactions are complete
